# 🐉 Dragon Ball API  

Esta é uma atividade desenvolvida para praticar o consumo de **APIs utilizando JavaScript**.

A aplicação permite que o usuário clique em um botão para **ativar a camera tirar uma foto para fazer uma analise ficticia **, exibindo seu **nome(personagem), ki e imagem** na tela. Outra funcionalidade é calcular a distancia da sua localização atual com a casa do Mestre Kami.

## ⚙️ Funcionalidade

Integração com Câmera (MediaDevices API): Acesso em tempo real à câmera do dispositivo para simular o "escaneamento" do usuário, capturando frames e manipulando-os via Canvas API.

Geolocalização Estilizada: Uso da Geolocation API para verificar a posição do usuário e realizar um cálculo lúdico da distância até a icônica Kame House.

Consumo de API Externa: Integração assíncrona (fetch) com a Dragon Ball API para buscar dados reais de mais de 50 personagens, incluindo imagens e níveis de KI.

Lógica de Análise de Poder: Algoritmo que ordena personagens por poder de luta e seleciona aleatoriamente o perfil que mais se adequa ao "KI detectado" na foto.

Manifesto de Aplicativo: Configuração completa de cores de tema, orientação de tela (portrait) e atalhos (shortcuts) para acesso rápido à câmera.
